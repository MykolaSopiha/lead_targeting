				!! ПРОЧТИ ЭТО !!

Даный скрипт позволяет настроить таргетинг на нужное гео и с нужных устройств.

				Инструкция:
						      
1. Папки:
	"good_site" - в эту папку нужно закинуть белый безпалевный сайт.
	"bad_site"  - ну ты понял)) тут твой ленд с хуями.

2. Настройки:
	Откройте файл "settings.php" и заполните код нужной страны в поле "country_code".
	Например, 
		
		'country_code' => "US",
	
	означает, что всем пользователям, которые не из США, будет показыватся белый ленд из
	папки "good_site".
	
	Дальше выберите настройки таргетинга проставляя true или false.
	Например,
	
		'Mobile' => false,
        'Tablet' => false,
        'Desktop' => true, 

	означает, что bad_site будет виден только на ПК, на мобильных устройствах и таблетах будет виден good_site.
